# Алгоритмы поиска кратчайшего пути на сеточном графе

## Описание

Программа реализует два классических алгоритма поиска кратчайшего пути на сеточном графе (решетке):

- **Алгоритм Форда-Беллмана** — пошаговое вычисление расстояний справа налево, сверху вниз
- **Алгоритм Минти** — вариант Дейкстры для сеточных графов

После вычислений программа визуализирует граф с найденным кратчайшим путем и отображает все промежуточные значения расстояний.

---

## Как работает

1. Пользователь задает размеры сетки (строки и столбцы)
2. Выбирает метод поиска: Форд-Беллман или Минти
3. Вводит веса ребер:
   - горизонтальные ребра (между соседними ячейками в строке)
   - вертикальные ребра (между ячейками соседних строк)
4. Программа пошагово вычисляет кратчайшие расстояния от верхнего правого угла до всех узлов
5. Восстанавливает путь из нижнего левого угла
6. Строит визуализацию: узлы с их расстояниями, ребра с весами, найденный путь выделен цветом

---

## Ввод данных

### Размеры
```
Сколько строк? 3
Сколько столбцов? 3
```

### Выбор метода
```
1 — Форд-Беллман
2 — Минти (Дейкстра)
```

### Веса ребер
Числа вводятся одним блоком в порядке:

- горизонталь строки 1 (cols-1 чисел)
- вертикаль строки 1-2 (cols чисел)
- горизонталь строки 2 (cols-1 чисел)
- вертикаль строки 2-3 (cols чисел)
- ...
- горизонталь строки rows (cols-1 чисел)

**Пример для сетки 3x3:**
```
4 6
1 2 3
5 7
4 5 6
8 9
```

Где:
- `4 6` — горизонтальные ребра первой строки
- `1 2 3` — вертикальные ребра между 1-й и 2-й строкой
- `5 7` — горизонтальные ребра второй строки
- `4 5 6` — вертикальные ребра между 2-й и 3-й строкой
- `8 9` — горизонтальные ребра третьей строки

## Визуализация

Программа строит граф, где:
- 🔵 **Синий узел** — цель (верхний правый угол, расстояние 0)
- 🟣 **Фиолетовые узлы** — найденный кратчайший путь и стартовая точка
- ⚪ **Серые узлы** — остальные вершины с вычисленными расстояниями
- **Фиолетовые ребра** — ребра, входящие в кратчайший путь
- **Серые ребра** — остальные ребра с подписанными весами
